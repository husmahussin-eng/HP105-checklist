# Meeting Minutes Documents

This folder contains PDF files for meeting minutes that are linked from the "Minit Mesyuarat" tab in the calendar viewer.

## Required PDF Files

Please upload the following PDF files to this folder:

1. `Minit meeting 01 05.11.2025.pdf`
2. `Minit meeting 02 12.11.2025.pdf`
3. `Minit meeting 03 25.11.2025.pdf`
4. `Minit meeting 04 3.12.2025.pdf`
5. `Minit meeting 05 10.12.2025.pdf`

## File Naming

The file names must match exactly as shown above (case-sensitive). These names correspond to the meeting documents listed in the static calendar viewer.

## How to Add Files

1. Copy your PDF files to this `documents/` folder
2. Make sure the file names match exactly
3. Commit and push to GitHub:
   ```bash
   git add documents/*.pdf
   git commit -m "Add meeting minutes PDF files"
   git push
   ```

## GitHub Pages

Once uploaded, GitHub Pages will automatically serve these PDF files, and users can download them by clicking the document links in the "Minit Mesyuarat" tab.
